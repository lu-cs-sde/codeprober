CodeProber

CodeProber is a live exploration tool for program analysis results. It has been specifically integrated with JastAdd, but can be used with other tools/metacompilers as well.

Code

CodeProber is open source, and available on Github at https://github.com/lu-cs-sde/codeprober/.

Usage

Please refer to the README for instructions on how to download and run CodeProber.

Demo

CodeProber was displayed at The Ninth Workshop on Live Programming, at ACM SIGPLAN SPLASH, Cascais, Portugal, 2023. The video is available on YouTube at: https://www.youtube.com/watch?v=_ZTv_Li1f_g.
